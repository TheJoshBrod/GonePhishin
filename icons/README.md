# Icons

Place PNG icons here:
- icon16.png  (16×16)
- icon48.png  (48×48)
- icon128.png (128×128)

You can generate them from an SVG with ImageMagick:
  magick icon.svg -resize 16x16 icon16.png
  magick icon.svg -resize 48x48 icon48.png
  magick icon.svg -resize 128x128 icon128.png

Or use any 🛡️ shield image as the base.
